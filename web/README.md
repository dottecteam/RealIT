# Real IT - WEB

Esta é a camada de interface do usuário do projeto Real IT, responsável por exibir os dados processados e persistidos.

---

## Como Executar o Projeto

Siga os passos abaixo para configurar o ambiente localmente:

### 1. Pré-requisitos

Certifique-se de ter o Node.js instalado na sua máquina (recomendado v20 LTS ou superior).

### 2. Instalação de Dependências

No diretório ```/web```, execute:

```bash
npm install
```

### 3. Executar em modo de Desenvolvimento

Para rodar o projeto, utilize:

```bash
npm run dev
```

O servidor estará disponível em: ```http://localhost:3000```.
