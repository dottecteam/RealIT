# Documento - Biblioteca do Mapa do Brasil

## 1. Fonte dos Dados
* **Nome:** mapa-brasil 
* **Link para acesso e download:** https://github.com/clagomess/mapa-brasil
* **Autor:** Cláudio Gomes
* **Linguagem utilizada:** `JavaScript`

---

## 2. Justificativa
A biblioteca, disponibilizada no GitHub, se destaca pela disponibilização de arquivos `.svg` para a construção <br>
visual do mapa dinâmico do Brasil. Além disso, oferece recursos que permitem a personalização completa dos arquivos, <br>
tornando a experiência do usuários mais agradável. <br>

Ademais, a biblioteca possui diferentes segmentos geográficos que possibilitam a visualização e manipulação de diversas <br>
maneiras, mostrando-se completa para utilização. Estão disponíveis:

* **Federação:** Unidades Federativas do Brasil.
* **Microrregiões:** Agrupamentos de municípios limítrofes com características geográficas, sociais e econômicas semelhantes.
* **Mesorregiões:** Agrupa municípios com características sociais, físicas e econômicas homogêneas dentro de um estado.
* **Municípios:** Menor unidade administrativa autônoma do Brasil, abrangendo áreas urbanas e rurais dentro de cada UF.
