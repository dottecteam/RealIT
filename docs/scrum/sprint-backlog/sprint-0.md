# Sprint 0 — Preparação e Documentação

---

| ID | Épico | História de Usuário | Critério de Aceite |
|----|-------|---------------------|--------------------|
| **00-1** | E0 | Como PO, quero definir e documentar as políticas de trabalho da equipe (regras de branch no GitHub, fluxo de commits, e critérios de permanência) para garantir o alinhamento, a organização do código e o comprometimento de todos. | Documento de diretrizes (ex: `git.md`) criado no repositório com as regras do Git, critérios de permanência definidos e cientes por toda a equipe. |
| **00-2** | E0 | Como PO, quero construir o Product Backlog inicial, estruturando os épicos e histórias de usuário, para que o time tenha visibilidade de todas as entregas planejadas para o MVP. | Backlog documentado na ferramenta de gestão (ex: GitHub Projects, Jira), com a Sprint 1 priorizada e os critérios de DoR e DoD definidos. |
| **00-3** | E0 | Como desenvolvedor, quero mapear e documentar os endpoints ou jsons relevantes da API do Banco Central para que a equipe saiba exatamente quais dados estão disponíveis para o cálculo do Índice de Oportunidade. | Documento com os endpoints mapeados, exemplos de requisição/resposta, formato dos dados e possíveis erros. |
| **00-4** | E0 | Como analista, quero elaborar a modelagem de Processos de Negócio utilizando as técnicas VPC (Value Proposition Canvas) e BPMN para garantir que a equipe compreenda o valor do produto para a DM. | Diagramas VPC e BPMN finalizados, validados pelo PO e anexados à documentação do projeto. |
| **00-5** | E0 | Como desenvolvedor backend, quero criar a modelagem do banco de dados relacional e a arquitetura lógica do software para definirmos a estrutura exata onde os dados processados do Banco Central serão salvos. | Diagrama Entidade-Relacionamento (DER) e diagrama de arquitetura lógica elaborados e salvos no repositório. |
| **00-6** | E0 | Como desenvolvedor frontend, quero montar o protótipo do dashboard no Figma utilizando bibliotecas de componentes prontos para validarmos o fluxo das telas e a disposição do ranking regional. | Protótipo navegável com a tela principal desenhada, aprovado pelo PO e com link compartilhado. |
